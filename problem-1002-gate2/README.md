# Problem 1002 二输入逻辑门

## 题目描述
包含下图所示的6个逻辑门：与门、与非门、或门、或非门、异或门、同或门。

![Problem 1002 二输入逻辑门的题目描述][description]

## 输入描述
输入a, b

## 输出描述
输出6位的 z

## 输入例子
a= 0, b = 1

## 输出例子
z = 6'b011010
