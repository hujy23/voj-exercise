# Problem 1000 ALU指令译码电路

## 题目描述
设计一个指令译码电路。该电路通过对指令的判断，对输入数据执行相应的操作，包括加、减、与、或和求反。

## 输入描述
输入为两个八位信号， 三位指令信号 opcode[000, 001, 010, 011，100] 分别代表加、减、与、或和求反，时钟周期：100ns

##输出描述
输出信号 out[7:0] 为 ALU 运算结果。

## 输入例子
a = 00000010, b = 00000011, opcode = 000.

## 输出例子
out = 00000101
